Története

Kialakulása

A 90-es években a Microsoft egy saját Java keretkörnyezetet kibővített saját függvényeivel és szolgáltatásaival.

Erre viszont nem volt engedélyük és szembe ment Java platformfüggetlen alapelvével ugyanis a Microsoft saját bővítményei miatt más rendszereken nem futott volna.

Ennek következményeként a Microsoft elkezdett fejleszteni egy saját keretrendszert a .NET-et (dotnet). Ennek a keretrendszernek volt az első nyelve C# (szísárp). Anders Hejlsberg volt a C# fejlesztés vezetője.

.NET

A fejlesztés a 90-es évek végén indult el Next Generation Windows Services címen. A .NET keretrendszer első verziója, a .NET Framework 1.0, 2002-ben jelent meg.

A .NET keretrendszer két fő részből áll: a Common Language Runtime (CLR) és a Framework Class Library (FCL):

  • CLR: Egy virtuális gép, amely futtatja a .NET nyelveken írt kódot. Ez felelős a kód fordításáért, futtatásáért és a memóriakezelésért.
  • FCL: Egy hatalmas könyvtár, amely különböző funkciókat és szolgáltatásokat biztosít, például adatbázis-kezelést, hálózati kommunikációt és grafikai megjelenítést.
Név eredete

A "C#" név eredete érdekes. Sokan úgy gondolják, hogy a "#" jel a C++-hoz kapcsolódik, mivel hasonlít a két "+" jele "egymásra csúsztatásához". Valójában a "#" jel a zenei félhangokat szimbolizálja, mivel az ASCII karakterkészletben nincs megfelelő jel számára.

A "C#" nyelv alapjaiban a C-hez kapcsolódik, de nem nyújt ugyanolyan nagy szabadságot, mint a C++, így valahol a két nyelv között helyezkedik el. A "#" jel pedig ezt a "C"-t emeli "fél hanggal" feljebb, hasonlóan a C++ "++" operátorához.

Fejlődése

A C# nyelv az évek során számos új funkcióval és fejlesztéssel bővült. A cél az volt, hogy a nyelvet még hatékonyabbá, könnyebben használhatóvá és rugalmasabbá tegyék.

Nyelvcsalád

C

C nyelvet Dennis Ritchie fejlesztette ki az 1970-es években. Ez az OG C.

C++

Ez a C nyelv kiterjesztése, ami objektumorientált programozást lehetővé teszi.

A ++ a programozásban egy használt rövidítést jelent, tehát C++ egyenlő C = C + 1-el.

C#
Történetéről részletesebben fenntebb. Jellemzőiről részletesebben lenntebb.
Rust

Fejlesztése 2006-ban indult a Mozilla Researchnál (Firofox készítője). Az első stabil verziót, a Rust 1.0-t 2015-ben adták ki.

Sok nagy cég használja például: Dropbox, Microsoft stb. A Firefox 10%-a Rustban írodott.

Go (Golang)

A Go nyelvet a Google mérnökei, Robert Griesemer, Rob Pike és Ken Thompson fejlesztették ki. A tervezés 2007-ben kezdődött, és a nyilvános bétaverziót, a Go 1-et 2012-ben adták ki.

Kotlin

A Kotlin nyelvet a JetBrains nevű szoftverfejlesztő cég fejlesztette ki. A Kotlin projekt 2011-ben indult, és az első nyilvános verzió 2016-ban került ki

Már a top 1000 androidos alkalmazásból 950 Kotlin-t használ Java helyett és a profi alkalmazások fele kotlint használ.
